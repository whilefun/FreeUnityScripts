+-------------------------+
| Editor Camera Bookmarks |
+-------------------------+


Description:
-----------
A simple editor script for Unity that allows you to use Ctrl+Number and Number to set and restore SceneView camera bookmarks.


How to Use:
----------

1) Put the EditorCameraBookmarks folder into your Assets folder.
2) With SceneView active, press Left Ctrl + a number to set a bookmark.
3) With SceneView active again, press the number from 2 to move the scene camera back to that bookmark
