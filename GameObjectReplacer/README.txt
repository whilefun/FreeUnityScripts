+----------------------+
| Game Object Replacer |
+----------------------+


Description:
-----------
A simple editor script that allows you to replace multiple game objects (non-prefabs or old prefabs) in your scene with a single replacement object (prefab)


How to Use:
----------

1) Select one or more Objects in your scene you want to replace
2) Select Tools -> Replace GameObjects
3) Focusing on the Window will update the selected array of Objects with your scene hierarchy selection.
4) Choose a replacement object (e.g. a prefab from the Project Explorer)
5) Press the "Replace..." button to replace the objects
6) If you need to undo, press Ctrl+Z or go to Edit->Undo.
