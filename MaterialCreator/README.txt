+------------------+
| Material Creator |
+------------------+


Description:
-----------
A simple script that creates Materials from selected Textures.


How to Use:
----------

1) Put the MaterialCreator folder into your Assets folder.
2) Select one or more Textures (e.g. in Assets/MyProjectFolder/Textures/)
3) Select Tools -> Create Materials From Selected Textures

The Textures will be created in a SIBLING folder called [MATERIAL_FOLDER_NAME] (Default: "Materials")
