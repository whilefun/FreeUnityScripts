+------------------------+
| Random Sound Container |
+------------------------+


Description:
-----------
A tiny set of scripts that allow you to play a random sound from a set of sounds, with randomized pitch and volume.


How to Use:
----------

To create a Random Sound Container:

1) Right click in Project and select Create > While Fun Games > Simple Random Sound Container
2) Select your new Simple Random Sound Container, and name it something like "MyCoolSoundContainer"
3) Add a set of sounds in the Clips area of the Inspector
4) Set the Volume and Pitch ranges
5) Click the Preview button in the Inspector, and repeat step 4 until you're happy with how it sounds

To use in your game, refer to the included script ExampleRandomSoundPlayer.cs, and example Prefab ExampleRandomSoundPlayer.prefab.
